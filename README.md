# Rebook
Este proyecto utiliza:

Angular versión 20.3.4

Bootstrap 5.3.8

FontAwesome


## Entorno de desarrollo

Al descargar el repositorio hay que instalar los módulos de npm con:
```bash
npm install
```

Después, se peude levantar el proyecto en `http://localhost:4200/`. con
```bash
ng serve
```

Si quieres que te abra automáticamente una ventana de tu navegador por defecto usa:
```bash
ng serve -o
```
